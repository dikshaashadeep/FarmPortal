# 🐄 SMART-AMU — Digital Farm Management Portal

> A web-based platform to monitor Antimicrobial Usage (AMU) and 
> Maximum Residue Limits (MRL) in livestock farms across India.

## 🚀 Project Overview
SMART-AMU helps farmers, veterinarians, and administrators track 
antibiotic usage in animals, calculate safe withdrawal periods, 
and check MRL compliance — ensuring food safety for consumers.

## 👥 Team
5 Members | GL Bajaj Group of Institutions | Internship Project 2025

## ✨ Features
- 🔐 Role-based Login (Farmer / Vet / Admin)
- 🐮 Animal Registration Module
- 💊 Medicine Entry & Treatment Logs
- ⏱️ Automatic Withdrawal Period Calculator
- 🟢🔴 MRL Compliance Checker (Safe/Unsafe Indicator)
- 📊 Interactive Dashboard with Charts
- 🔔 Alert System for restricted sale periods
